# Spirithought 数据库管理文档

## 一 数据表

### 1. 文档（随笔）

```javascript
{
  _id: "ObjectId('...')",    // 默认主键
  name: "文章名，标题",
  directoryId: "所属目录id",
  directoryName: "所属目录名",
  author: "作者",
  createTime: "创建时间",
  description: "简介，用于列表的简短显示",
}
```

### 2. 目录

```javascript
{
  _id: "ObjectId(...)",   // 默认主键
  name: "目录名",
  creator: "创建人",
  createTime: "创建时间",
  description: "目录简介"
}
```

### 3. 用户

```javascript
{
  _id: "ObjectId(...)",    // 默认主键
  username: "用户名"
  pwd: "登录密码哈希值",
  createTime: "注册时间",
}
```
